فحص Firebase Realtime Database

هذه النسخة لا تستخدم Firebase Auth ولا Firestore. الحفظ والطلبات والطاولات كلها على Realtime Database.

تسجيل الدخولالمنيو

قواعد Realtime Database المطلوبة

{ "rules": { "restaurants": { "r7_burger": { ".read": true, ".write": true } } } }

انسخ القواعد إلى: Firebase Console > Realtime Database > Rules ثم Publish. بدونها لن يتم الحفظ.
